# Discord-bot

Discord bot who aim is to play songs or playlists from yotube or spotify.<br/>
To add bot on your server use this [link](https://discord.com/oauth2/authorize?client_id=927270836631650335&permissions=0&scope=bot%20applications.commands)<br/>
If you just test it you can go on this [server](https://discord.gg/XxMjAGk3JK) 
Stack of technologies:
- Node js
- Discord.js
- Distube
- Deployed on Heroku

P.S Bot can't play private playlist only public
